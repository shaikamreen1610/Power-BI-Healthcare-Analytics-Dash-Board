
# Healthcare Analytics Dashboard | Power BI
## Overview

The Healthcare Analytics Dashboard is an end-to-end Power BI solution designed to provide actionable insights into hospital operations, doctor performance, patient management, billing, commissions, and medicine usage.
The dashboard enables hospital stakeholders to monitor key metrics, optimize revenue, and improve operational efficiency through data-driven decision-making.

## Business Objectives

Analyze doctor performance, availability, and commissions

Track patient lifecycle (admission to discharge)

Monitor hospital revenue and billing patterns

Evaluate medicine consumption and sales trends

Provide a centralized view of healthcare KPIs
